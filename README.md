# OutreachLog

**Simple cold outreach tracker for solopreneurs.**

Stop guessing which cold emails work. Track everything. See what converts.

---

## 🎯 What It Does

OutreachLog helps solopreneurs track their cold outreach across email, LinkedIn, and Twitter.

**Features:**
- ✅ Track unlimited contacts (Pro plan)
- ✅ Save & test message templates
- ✅ See response rates per template
- ✅ Follow-up reminders
- ✅ Analytics dashboard
- ✅ CSV import/export
- ✅ ROI calculator

**Built for:**
- Coaches, consultants, freelancers
- Early-stage founders (pre-CRM budget)
- Sales folks with side hustles
- Anyone doing cold outreach

## 📊 How It Works

### 1. Add Contacts
Track who you're reaching out to (name, email, company, source).

### 2. Save Templates
Save your best cold email/DM templates. Track which ones get responses.

### 3. Log Outreach
Log every email/DM you send. Mark when they respond.

### 4. See Analytics
- Overall response rate
- Response rate by template
- Outreach by channel (email vs LinkedIn vs Twitter)
- Conversion tracking

### 5. Improve
Clone your best-performing templates. Test variations. Increase response rates.
