## Implementation with AgentMail

**Setup:**

1. Create AgentMail account: https://www.agentmail.to
2. Get API key: Dashboard → API Keys
3. Create email sequences:
   - Sequence 1: Welcome & Onboarding (Days 0, 1, 3, 7, 14)
   - Sequence 2: Re-engagement (Day 21 if inactive)
   - Sequence 3: Pro Onboarding (immediate post-purchase)

4. Trigger logic:
   - User signs up → Sequence 1 starts
   - User upgrades → Sequence 3 starts, Sequence 1 stops
   - User inactive 14 days → Sequence 2 starts

5. Personalization variables:
   - {firstName} - User's name
   - {contactCount} - Current contact count
   - {responseRate} - Current response rate (if available)

**Code example (webhook trigger):**

```javascript
// When user signs up (LocalStorage → AgentMail)
async function sendWelcomeEmail(userEmail, userName) {
    await fetch('https://api.agentmail.to/v1/messages', {
        method: 'POST',
        headers: {
            'Authorization': 'Bearer YOUR_API_KEY',
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({
            to: userEmail,
            subject: 'Welcome to OutreachLog! Here\'s how to get started',
            text: `Hey ${userName},\n\nWelcome to OutreachLog!...`,
            scheduledFor: new Date().toISOString() // Send immediately
        })
    });
}

// Schedule Day 1 email
async function scheduleDay1Email(userEmail, userName) {
    const tomorrow = new Date();
    tomorrow.setDate(tomorrow.getDate() + 1);
    
    await fetch('https://api.agentmail.to/v1/messages', {
        method: 'POST',
        headers: {
            'Authorization': 'Bearer YOUR_API_KEY',
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({
            to: userEmail,
            subject: '3 tips to get more from OutreachLog',
            text: `Hey ${userName},\n\nQuick check-in!...`,
            scheduledFor: tomorrow.toISOString()
        })
    });
}
```
